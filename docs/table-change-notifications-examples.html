Monitor table change examples

Receive SQL Server notifications on record table change with C#

Here are two examples of application that receive notifications on record table changes using SqlTableDependency nuget package.

Detect table changes with WPF and WCF

Source code available at https://github.com/christiandelbianco/Monitor-table-change-with-WPF-WCF-sqltabledependency.

This example show how to keep up to date a grid containing some stocks data.
That grid has been automatically updated whenever a record change using SQL Server notifications.
This notification contains new values for the modified table record change.

Audit table change with ASP.NET MVC, SignalR and Knockout JS

Source code available at https://github.com/christiandelbianco/monitor-table-change-with-mvc-signalR-knockoutjs-sqltabledependency.

This example show how to refresh client web browsers used to book flight tickets.
Those terminals have to be update as soon as the availability change and the Web application must take the initiative of sending this information to clients instead of waiting for the client to request it.
